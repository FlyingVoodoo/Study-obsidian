# Билет 7. Свободные незатухающие электромагнитные колебания

> [!info] Формулировка вопроса

## 1. Понятие квазистационарных токов и электромагнитных колебаний

* **Электромагнитные колебания** — это периодические (или квазипериодические) изменения взаимосвязанных характеристик электрического и магнитного полей (заряда, напряжения, силы тока, напряженности и индукции).
* **Условие квазистационарности:** При рассмотрении электрических колебаний применяются законы постоянного тока (закон Ома, правила Кирхгофа) для *мгновенных значений* величин. Это справедливо, если время распространения электромагнитного сигнала по цепи $t = l/c$ (где $l$ — длина цепи, $c$ — скорость света) много меньше периода колебаний $T$:
  $$t \ll T \implies \frac{l}{c} \ll T$$
  Токи, удовлетворяющие этому условию, называются **квазистационарными**. Их мгновенное значение силы тока во всех последовательных участках цепи практически одинаково.

---

## 2. Идеальный колебательный контур и физика процесса

**Колебательный контур** — это электрическая цепь, состоящая из последовательно соединенных катушки индуктивности $L$ и конденсатора ёмкостью $C$. 

Контур называется **идеальным**, если его активное сопротивление $R = 0$. В такой модели отсутствуют потери энергии на нагревание проводников (выделение джоулева тепла), поэтому свободные колебания являются *незатухающими*.



### Физический механизм колебаний (периодический процесс):
1. **Начальный момент ($t=0$):** Конденсатор заряжен до максимального заряда $q_m$ (разность потенциалов $U_m$). Ток в цепи равен нулю ($I=0$). Вся энергия сосредоточена в электрическом поле конденсатора.
2. **Разрядка конденсатора ($0 < t < T/4$):** Конденсатор начинает разряжаться, в катушке появляется ток $I$. Ток растёт плавно, так как ему препятствует возникающая ЭДС самоиндукции катушки $\varepsilon_{si}$. Электрическое поле убывает, магнитное поле катушки растёт.
3. **Момент $t = T/4$:** Конденсатор полностью разряжен ($q=0$). Сила тока достигает максимального значения $I_m$. Вся энергия переходит в магнитное поле катушки.
4. **Перезарядка ($T/4 < t < T/2$):** Ток в силу электромагнитной инерции (самоиндукции) не может исчезнуть мгновенно. Он продолжает течь в прежнем направлении, уменьшаясь по модулю, и переносит заряды на противоположные обкладки конденсатора.
5. **Момент $t = T/2$:** Ток падает до нуля ($I=0$), а конденсатор полностью перезаряжается (знаки зарядов обкладок противоположны начальным). Процесс повторяется в обратную сторону.

## 3. Дифференциальное уравнение свободных колебаний

Для вывода уравнения запишем закон Ома для мгновенных значений квазистационарного тока в замкнутом контуре. Внешние источники ЭДС отсутствуют, следовательно, падение напряжения на конденсаторе $U_C$ в любой момент времени уравновешивается ЭДС самоиндукции катушки $\varepsilon_{si}$:
$$U_C = \varepsilon_{si}$$

Выразим эти величины через мгновенный заряд конденсатора $q$:
* Напряжение на конденсаторе: $U_C = \frac{q}{C}$
* ЭДС самоиндукции в катушке: $\varepsilon_{si} = -L \frac{dI}{dt}$

Подставляем в исходное соотношение:
$$\frac{q}{C} = -L \frac{dI}{dt} \implies L \frac{dI}{dt} + \frac{1}{C}q = 0$$

По определению сила тока — это скорость изменения заряда со временем: $I = \frac{dq}{dt} = \dot{q}$. Следовательно, производная тока по времени есть вторая производная от заряда по времени: $\frac{dI}{dt} = \frac{d^2q}{dt^2} = \ddot{q}$.

Подставляя $\ddot{q}$ в уравнение и разделяя на $L$, получаем **каноническое дифференциальное уравнение свободных незатухающих колебаний**:
$$\ddot{q} + \frac{1}{LC} q = 0 \quad \text{или} \quad \frac{d^2q}{dt^2} + \omega_0^2 q = 0$$

Где величина $\omega_0$ — **собственная циклическая частота** колебательного контура:
$$\omega_0 = \frac{1}{\sqrt{LC}}$$

Из связи циклической частоты с периодом ($T = \frac{2\pi}{\omega_0}$) следует **формула Томсона** для периода свободных незатухающих колебаний:
$$T = 2\pi\sqrt{LC}$$

## 4. Решение дифференциального уравнения

Решением линейного однородного дифференциального уравнения второго порядка является гармоническая функция (в форме косинуса):
$$q(t) = q_m \cos(\omega_0 t + \alpha_0)$$

Где:
* $q_m$ — амплитуда колебаний заряда конденсатора;
* $\omega_0 t + \alpha_0$ — фаза колебаний в момент времени $t$;
* $\alpha_0$ — начальная фаза (зависит от выбора момента начала отсчета времени $t=0$).

### Колебания напряжения $U(t)$:
$$U(t) = \frac{q(t)}{C} = \frac{q_m}{C} \cos(\omega_0 t + \alpha_0) = U_m \cos(\omega_0 t + \alpha_0)$$
*Амплитуда напряжения:* $U_m = \frac{q_m}{C}$. Колебания напряжения происходят синфазно с колебаниями заряда.

### Колебания силы тока $I(t)$:
Сила тока находится как первая производная от заряда по времени:
$$I(t) = \frac{dq}{dt} = -q_m \omega_0 \sin(\omega_0 t + \alpha_0) = I_m \cos\left(\omega_0 t + \alpha_0 + \frac{\pi}{2}\right)$$
*Амплитуда силы тока:* $I_m = q_m \omega_0 = U_m \sqrt{\frac{C}{L}}$.

> **Важный теоретический вывод:** Колебания силы тока опережают по фазе колебания заряда и напряжения на $\frac{\pi}{2}$ (четверть периода). В моменты максимального заряда конденсатора ток равен нулю, а в моменты полной разрядки ток достигает максимума.

## 5. Энергия колебаний и закон её сохранения

Электромагнитные колебания в контуре сопровождаются поочередным превращением энергии электрического поля в энергию магнитного поля.

* **Мгновенная энергия электрического поля конденсатора:**
  $$W_E(t) = \frac{q^2(t)}{2C} = \frac{q_m^2}{2C} \cos^2(\omega_0 t + \alpha_0)$$
* **Мгновенная энергия магнитного поля катушки:**
  $$W_M(t) = \frac{L I^2(t)}{2} = \frac{L I_m^2}{2} \sin^2(\omega_0 t + \alpha_0)$$

Подставляя значение амплитуды тока $I_m^2 = q_m^2 \omega_0^2 = q_m^2 \frac{1}{LC}$, запишем магнитную энергию как:
$$W_M(t) = \frac{L}{2} \left(\frac{q_m^2}{LC}\right) \sin^2(\omega_0 t + \alpha_0) = \frac{q_m^2}{2C} \sin^2(\omega_0 t + \alpha_0)$$

### Закон сохранения полной энергии:
Полная энергия электромагнитного поля контура $W$ в любой момент времени равна сумме электрической и магнитной энергий:
$$W = W_E(t) + W_M(t) = \frac{q_m^2}{2C} \cos^2(\omega_0 t + \alpha_0) + \frac{q_m^2}{2C} \sin^2(\omega_0 t + \alpha_0)$$

Используя основное тригонометрическое тождество ($\cos^2 x + \sin^2 x = 1$), получаем постоянство полной энергии:
$$W = \frac{q_m^2}{2C} = \frac{C U_m^2}{2} = \frac{L I_m^2}{2} = \text{const}$$

**Вывод:** Полная энергия идеального колебательного контура сохраняется неизменной во времени. Происходят взаимные превращения энергий полей, при этом частота колебаний энергии в два раза выше частоты колебаний заряда и тока ($\omega_{energy} = 2\omega_0$), а средние за период значения электрической и магнитной энергий равны между собой: $\langle W_E \rangle = \langle W_M \rangle = \frac{W}{2}$.
